---
title : 'Paragraph'
tag : '<p>'
---

- `<p>` html element represents a paragraph.
- Paragraph as block of text.

## Attributes

Paragraphs only includes the global attributes.

## Paragraph Example

### HTML

```html
<p>
    Many tags, particularly the closing end tag for the very commonly used paragraph element <p>, are optional. An HTML browser or other agent can infer the closure for the end of an element from the context and the structural rules defined by the HTML standard.
</p>
```

### Paragraph with Attribute

```html
<p style="color:grey">
    Many tags, particularly the closing end tag for the very commonly used paragraph element <p>, are optional.
</p>
```

### Paragraph with CSS

```html
<p>
    Many tags, particularly the closing end tag for the very commonly used paragraph element <p>, are optional.
</p>
```

```css
p {
    color: 'black';
    font-size: '1rem';
}
```
